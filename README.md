## 💡 Why I Made This

<div align="center">

*"I believe cybersecurity knowledge should be accessible to everyone — not locked behind paywalls."*

</div>

When I started my journey in cybersecurity, I spent **countless hours** searching the internet for quality learning material. Resources were scattered across random websites, some behind paywalls, others buried in obscure forums. I wished there was **one single place** where I could find everything I needed — organized, categorized, and ready to use.

**So I built it myself.**

This repository is the resource I wish I had when I started. It's for:

- 🎓 **Students** who can't afford expensive courses but are hungry to learn
- 💼 **Career switchers** who need a solid foundation before their first security role
- 🔬 **Researchers** who need quick access to reference material
- 🏆 **Certification seekers** preparing for CEH, OSCP, CISSP, CompTIA, and more
- 🌍 **The global community** — especially those in regions where access to quality resources is limited

> **My goal is simple:** Remove barriers to cybersecurity education and help the next generation of security professionals get started — no matter where they are or what their background is.

```
📁 This repo is your structured learning companion.
    Follow this order for optimal progression:
```

### 🟢 Complete Beginner (0–3 months)

| Step | Action |
|:----:|--------|
| 1 | Follow [roadmap.sh/cyber-security](https://roadmap.sh/cyber-security) |
| 2 | Study CompTIA Network+ / Security+ ([Professor Messer YouTube](https://youtube.com/@professormesser)) |
| 3 | Complete TryHackMe **"Pre-Security"** and **"Complete Beginner"** paths |
| 4 | Practice on [OverTheWire Bandit](https://overthewire.org/wargames/bandit/) |
| 5 | Watch [NetworkChuck](https://youtube.com/@NetworkChuck) for Linux & networking fundamentals |
| 6 | Explore the [📚 Security Books Library](https://github.com/mizazhaider-ceh/Security-Books) for beginner PDFs |

### 🟡 Intermediate (3–12 months)

| Step | Action |
|:----:|--------|
| 1 | Complete TCM Security **"Practical Ethical Hacking"** course |
| 2 | Solve PicoCTF and root 10+ TryHackMe machines |
| 3 | Work through **PortSwigger Web Security Academy** (all topics) |
| 4 | Start Hack The Box Easy machines (reference [IppSec](https://youtube.com/@ippsec)) |
| 5 | Pursue **eJPT** or **PNPT** certification |

### 🔴 Advanced (12+ months)

| Step | Action |
|:----:|--------|
| 1 | Start Hack The Box **Pro Labs** (RastaLabs, Offshore) |
| 2 | Work through OffSec **Proving Grounds** for OSCP prep |
| 3 | Deep-dive into a specialty (Red Team, DFIR, Cloud, AppSec) |
| 4 | Contribute to **bug bounty** on HackerOne / Bugcrowd |
| 5 | Pursue **OSCP / CRTO / GREM / CISSP** |
